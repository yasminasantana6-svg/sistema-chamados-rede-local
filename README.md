## Explicação do sistema

O sistema corporativo permite o registro de chamados técnicos dentro de uma empresa.  
Os colaboradores podem informar nome, setor e solicitação do problema.

Os dados são enviados para uma API em Node.js, armazenados no MySQL e exibidos em tempo real no navegador.  
O sistema funciona em rede local, permitindo acesso entre computadores diferentes.
